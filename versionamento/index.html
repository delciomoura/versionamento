Projeto Versionamento

Sobre o Projeto

Este é um projeto da faculdade desenvolvido para explorar e verificar o uso de versionamento de código utilizando o Git. O objetivo principal é aprender e praticar as principais funcionalidades e práticas do versionamento de código, incluindo a criação de branches, commits, merges e pushs.

Como Usar

  1. Clone este repositório em sua máquina local usando o comando: 
    git clone https://github.com/seu-usuario/versionamento.git
  2. Navegue até o diretório do projeto: 
    cd versionamento
  3. Crie um branch para suas modificações: 
    git checkout -b feature/nova-feature
  4. Faça as alterações necessárias no código e adicione as modificações ao commit: 
    git add .
git commit -m "Adicionando nova feature"
  5. Envie suas alterações para o repositório remoto: 
    git push origin feature/nova-feature
  6. Crie uma solicitação de pull request para revisão e integração do seu branch de feature ao branch principal do projeto.

Contribuição

Contribuições são bem-vindas! Sinta-se à vontade para abrir uma issue caso encontre algum problema ou para enviar um pull request com melhorias.
